"Список задач"
Данный контракт имеет фукнции:

addTask
Добавляет задачу, время и флаг в сопоставлении с ключом.
getTask
Получает структуру по введенному ключу
getAllTask
Выводт все структуры
openTask
Выводит все невыполненные задачи(флаг = false)
flagChange
Отмечает что задача выполнена(по ключу)
removeTask
Удаляет задачу по ключу
